# Cheng Shang Portfolio

This is the portfolio website for Cheng Shang, showcasing projects, skills, and contact information.

## [Visit](https://cheng-shang.me)

## Run Locally

Install Node.js LTS and enable corepack, then run the following commands:

```sh
git clone git@github.com:shangc1204/portfolio.git
cd portfolio
pnpm i
pnpm dev
```
